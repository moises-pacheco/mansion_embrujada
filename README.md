# Mansión embrujada

He desarrollado una mansión embrujada interactiva para que el aprendizaje sea mucho más divertido, todo está hecho desde cero: la mansión, el árbol, las luciernagas, el terreno, todo está hecho con Three.js excepto el fantasma junto con la farola que fueron hechos con Blender. Es un proyecto que fue muy duro de realizar pero aprendí muchísimo con él, nadie quita que el día de mañana siga mejorando el código e implementando nuevas cosas a medida que siga aprendiendo.
